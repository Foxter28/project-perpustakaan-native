# Sistem Perpustakaan PHP Native

Sistem perpustakaan sederhana menggunakan **PHP Native** dan **MySQL** dengan tampilan **Bootstrap**.  
Sistem ini mendukung **login**, **CRUD anggota (Members)**, **Kategori**, dan **Buku**.
